This project is an Admin Dashboard layout built using HTML and CSS Grid as part of The Odin Project curriculum.

## 🚀 Live Demo

---

## 📌 Project Overview

The Admin Dashboard is a static layout project focused on mastering CSS Grid.

The layout consists of:

- Sidebar navigation
- Header with search bar and user controls
- Main content area
- Project cards
- Announcements section
- Trending section

This project emphasizes grid-based layout structuring and nested grids.

---

## 🛠️ Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox (for small sections)

## ✨ Features

- Full dashboard layout using CSS Grid

- Project card layout
